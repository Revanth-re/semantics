HTML


1."Hypertext markup language" is a language which is used to create static web pages with the help of built in tags.
2.Hypertext means collection of text and links.
3.Html is created by Tim Bernes Lee in 1990.
4.HTML5 is introduced in 2014
5.W3C[world wide web constrotium]organisation manages whole html language.
6.Keywords:Keywords consists of predefined logic written by language itself.
7.Syntax:rules given by language.
8..Webpage. refers to the collection of data which is connected together.

TAGS

TAGS are The keywords which enclosed between < and > symbols to display data in the browser.

TYPES OF TAGS

1.Paired tags or container tags.
2.Unpaired tags or empty tags.

1.PAIRED TAGS:The tags which is having opening and closing tag is known as paired tags.
exp:p,h1,div,span,table

2.UNPAIRED TAGS:The tags which doesnt have any closing tag,it closes itself
exp:img,br,input

___STRUCTURE OF AN HTML___

DOCTYPE SECTION
ROOT ELEMENT SECTION
HEAD SECTION
BODY SECTION

ELEMENTS

combination of opening ,closing tags and with content between the tags is known as elements.
1.opening tag
2.content
3.closing tag

TYPES OF ELEMENTS

1.BLOCK-LEVEL ELEMENTS
2.INLINE ELEMENTS
3.INLINE-BLOCK ELEMENTS

1.BLOCK-LEVEL ELEMENTS:The tag which occupies whole width and creates a new line.
exp:div,h1toh6,semantic elements

2.INLINE:Elements occupies a required space and doesnt creates a new line.
in css we cant apply width,height,margin top and bottom,padding top and bottom.
exp:span ,i,b,strong

3.INLINE-BLOCK ELEMENTS: The element which occupies only reqired space,doesnt creates a new line.
exp:img, iframe,input,button
