Structure of a URL


URL Structure[Universe resource location]


It is the address used to access the resource address.The structure of url is divided to three classes those are
Protocol
Domain
Path

 1.Protocol:
It is the set guidelines or rules helps us to display the data over the internet.There are different types in protocol mainly http.
2.Domain:
Domain make it easy for users to access websites without needing to remember IP addresses.There are two types in Domain.
a.Second Level Domain
b.Top Level Domain
 3.Path:
The Path specifies the location of the resource on the server

Agile methodology


The Azile methodolgy is used in every software company for higher efficiency because it conatins some methodology that is:
The project is firstly divided into subparts called as SPRINTS.
1.In first method they will divide the projects into sprints and they will assign that sprints to the developers.
2.And they will schedule the time to the developers such as 2 to 3 weeks.
3.While we are doing the sprint we will get SCRUM CALL
it means they will clarify doubts and they will track the sprint and they will clear if we have any problems in
the sprint.The scrum call will be managed by SCRUM MASTER.
4.After completion of sprint it will go to testing department .The Testing department will test if it is approved by the Testing department they will move to sprint 2.
5.If it is not approved then again it will go to sprint 1.
